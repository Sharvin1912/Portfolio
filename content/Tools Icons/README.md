# Tools Icons Folder

Place your tool icons here with serial names so cards map automatically by order.

Examples:
- icon1SolidWorks.png
- icon2Ansys.png
- icon3AbaqusCAE.png
- icon4Creo.png

How mapping works:
- `icon1*` goes to first tool card
- `icon2*` goes to second tool card
- `icon3*` goes to third tool card
- and so on

Supported formats:
- .png, .jpg, .jpeg, .webp, .svg

If an icon index is missing, the card falls back to a default placeholder icon.
